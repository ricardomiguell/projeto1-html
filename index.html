Surgimento do HTML

Voltar

O HTML (HyperText Markup Language) foi criado por Tim Berners-Lee em 1991 como parte do desenvolvimento da World Wide Web. A ideia era criar uma linguagem simples para formatar e organizar conteúdo na internet, permitindo a comunicação entre pesquisadores e universidades. Sua primeira versão foi bastante simples, com o objetivo de tornar a troca de informações mais acessível.

A evolução do HTML ao longo dos anos tem sido essencial para a criação de sites mais complexos. Enquanto no início ele apenas permitia a estruturação básica de textos e links, novas versões foram incorporando recursos para multimídia, formulários e interatividade. O HTML 5, lançado em 2014, trouxe importantes inovações, como suporte nativo para áudio e vídeo.

Hoje, o HTML é uma das linguagens mais importantes da web. Embora novas tecnologias tenham surgido, como o CSS e o JavaScript, o HTML permanece fundamental para a construção de páginas web, sendo usado como base para estruturar e organizar o conteúdo que vemos na internet.

Cliente e servidor

Voltar

Na arquitetura de redes, o modelo **cliente-servidor** descreve a relação entre dois tipos de sistemas: o cliente, que faz solicitações de serviço, e o servidor, que as processa e responde. No contexto da web, o navegador (cliente) solicita uma página web ao servidor, que então envia os dados necessários para exibir o site.

O cliente é responsável por interagir com o usuário e apresentar a interface, enquanto o servidor lida com o processamento de dados, como consultar um banco de dados ou armazenar informações. Esse modelo permite que vários clientes acessem a mesma aplicação de forma eficiente e segura.

Esse conceito é essencial para o funcionamento da internet e das aplicações web modernas, sendo a base para protocolos como HTTP, que define como as requisições e respostas entre cliente e servidor devem ser feitas.

O que é HTML

Voltar

HTML, ou **HyperText Markup Language**, é a linguagem de marcação usada para estruturar conteúdo na web. Ela define os elementos de uma página, como títulos, parágrafos, imagens e links. Cada elemento é representado por uma tag, como <h1> entre outras.

Ao contrário das linguagens de programação, o HTML não executa funções lógicas, mas organiza o conteúdo de forma que os navegadores possam exibi-lo corretamente. O HTML também permite a criação de links entre páginas, formando a base da navegação na web.

Embora o HTML seja uma linguagem estática, ele é amplamente utilizado em conjunto com outras tecnologias, como CSS e JavaScript, para criar páginas web interativas e dinâmicas, melhorando a experiência do usuário.

O que são tags HTML

Voltar

As **tags HTML** são elementos usados para estruturar e organizar o conteúdo de uma página web. Elas indicam ao navegador como o conteúdo deve ser exibido. Exemplos de tags incluem "h1" para títulos,"p" para parágrafos e "a" para links.

Cada tag HTML geralmente possui uma tag de abertura e uma de fechamento, como "div", mas algumas tags, como "img" e "br", são auto-fechantes. Essas tags são fundamentais para criar a hierarquia e a semântica do conteúdo na página.

Além disso, as tags podem incluir atributos, como `class` ou `id`, para fornecer mais controle sobre o estilo e a funcionalidade do conteúdo, facilitando a aplicação de CSS ou interações via JavaScript.
